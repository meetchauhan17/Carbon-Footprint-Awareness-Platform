# 🌱 Carbon Footprint Awareness Platform

## Challenge
[Challenge 3] Carbon Footprint Awareness Platform - Hack2Skill Prompt Wars

## Overview
A full-stack React PWA that helps individuals track, understand, and reduce their daily carbon footprint through personalized insights, gamification, and actionable eco tips.

## Features
- 📊 Multi-category carbon calculator (Transport, Energy, Food, Shopping)
- 📈 Interactive charts and history tracking
- 🏆 Badges and achievement system
- 💡 30+ actionable eco tips with completion tracking
- 🌐 Free API integrations (weather, quotes, country data)
- 📱 PWA — works offline, installable on mobile
- ♿ WCAG AA accessible

## Tech Stack
- React 18 + Vite
- React Router v6
- Recharts (data visualization)
- TailwindCSS (styling)
- LocalStorage (data persistence)
- Vitest + Testing Library (tests)

## Emission Factors Source
EPA Emission Factors Hub, IPCC AR6, Carbon Independent.org

## How to Run
```bash
npm install
npm run dev
```

## How to Test
```bash
npm test
```

## Assumptions
- Daily average global footprint used: 11kg CO2/day (4 tons/year)
- India electricity grid emission factor: 0.82 kg CO2/kWh (CEA 2023)
- All calculations are estimates for awareness purposes only
